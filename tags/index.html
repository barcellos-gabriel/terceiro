Uso de <span> para Estilo

A tag <span> é um container genérico em linha, útil para aplicar estilos a um trecho específico de texto, como esta palavra em roxo, sem afetar o resto do parágrafo.

Exemplo de Código Formatado

Para criar um link em HTML, usamos a tag <a>, como no exemplo abaixo:

<!-- Link simples para um site -->
<a href="https://www.google.com">Ir para o Google</a>
